So you ask, how did I manage to make this masterpiece of a font?
Well here's my step by step process during the making of this font!




Step 1: Creating the font

The first thing that I did was to open calligraphr, then creating the getting the template.
Using that template, I decided on what kind of font I would make.
It took me like 10 minutes to think of the idea, and then an hour and a half to fully make the font.




Step 2: Creating the website for the font

Once I had finally finished the font (Slide #1 above). I built and exported the font.
After that, I proceeded to create a html file and css file. This is where I made a decently big mistake.
What did was try to make parts of the site in html, then immediately use css to style it.
This worked for a little bit, but then it all came crashing down and everything broke (see slide 2)


Step 3: ACTUALLY creating the website (the correct way)

As a man who always learns from his mistakes (cough cough tony stark cough)
I restarted the website, and focused only on the html part. Helped me breeeze through the website.
I added most of the features I wanted to add, and also described how and what the font could be used for.
I also added boxes so that the user could test out typing stuff in the font too.
I actually really liked the font boxes lol (check slide 3 for state of site)
After this, I decided that I added enough to style the website I bit.
Looking at the font reminded me of something I would see in a notebook or a drawing of sorts.
So I decided to make the website notebook themed, which I did by making the background look like one (slide 4)
(you may also see an attempt of this on the failed first site on slide 1)

Step 4: Adding more ot the website

After that, I took a bit to think about what to add next.
Which led me to create this sort of doohickeys section.
Where it's a bunch of cool things related to the font, to show it off!
So I added another text (font) box, which I eventually gave a slider to change the font size.
Then I added a big box so the user could look at the font a lot lot closer.
I also added a section where I could show off every single character in the font I think my favourite characters would be $ and #
I later also added a section at the bottom to allow the user to download the font.

Step 5: Finding things to add

Around this point is when I started to get stuck for ideas.
I think I had around 7 hours at this point, and I was aiming to hit 15 hours for that ereader.
that's when I remembered that I could have more than one page lol. So I made a new page.
Currently I have 5 pages (main, story, creation procees "this one!",poster examples using the font, and one letter one song)
I'll go into each of them in depth in a second. But I want to look at the main page here for a bit longer.
As I started creating pages, I was also trying to add some music that would play while on the site.
So I started to add an audio player, and to make sure the user could access the controls, I added it to the top of the page
(You might be able to tell where this is going)
As I added the audio player to the top, I also needed a way for the user to download the font.
I added the download font section at the bottom of the site, so it wouldn't be in the way.
But I realised that scrolling all the way down for the font was kind of in the way.
So I made a quick link that would allow the user to quickly reach the bottom to the downloads section.
I also decided to place the link at the top, along with the audio player.
I ALSO needed a place to allow the user to access the new page(s).
So I also chucked the link(s) with both the audio player and the quick link to the downloads section.
At this point I also realised that the links were starting to get in the way lol.
Which gave me the bright idea to turn it into a header, similar to other websites that hide it when the user scrolls down.
Which I ended up doing. It was a bit of a hassle, but I managed to do implement it in the end. (See slide 6)

I kinda struggled to think of an idea for the first page.
All I could think of was what else to add to show off the font more. I looked closer at the font and realised something
I could make a story off this!!!!. What a magnificent idea I just thought of.

I thought of a story that was kinda like 2 society's that were already split up and how they were already kinda doing their own thing.
But that there was also a legend that both sides needed to come together to fullfil. This story is kind of like a fantasy lol.
It kinda looks like a plot summary for the start of a story, but I don't think I'll finish it unfortunately.
If someone really likes the story, then I'll consider writing a bunch more for it.
After I wrote the amount of story I have, I added in the styling using the same css as the main page, and finished the page!!
