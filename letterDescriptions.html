Here is every character in the font, and a description for every single one!

a - This letter could be used for a replacement for the famous :) face!

b - This one is a simpler version of :b, a very chill character!

c - If you've seen those big yellow smiling emoji on like kids tv shows. Then this would basically be that!

d - Similar to the previous one, this is a simpler version of d:

e - This could be used as a mocking face or when being devious.

f - This letter was inspired by that one facebook logo art I was saw a long back while ago. Could be used to show something related to doomscrolling or back pain lol

g - Now this is a cool font... get it? cause it has sunglasses on?.. Moving on, this is on the same level as the sunglasses emoji!

h - Now this is a funky letter. I would imagine someone using this when they see something they want to eat (like a pancake with ice cream)

i - I think this letter looks perfectly as it's used. It's an individual standing. Just like how people use the letter to refer themselves.

j - This is just another variation of i, except it's got a bit more curve it.

k - this is probably the most unique lowercase letter in the font. This letter could be used to present something great!

l - I see this letter as just "meh" or just a neutral expression.

m - This is just a :3 face but turned 45 degrees anti clockwise lol

n - This is also just c, but this time turned 45 degrees clockwise!

o - I wouls use this leter to show my shocked reaction at something. Like "Whaaat?! o"

p - the iconic :p face. Another simple letter to use.

q - same as p, but flipped horizontally

r - this one is interesting. It's a smiley face but... diagonal?!

s - now this is truly revolutionary! 2 smiley faces in one!!

t - kind of a mix of the T-pose, but also a person sitting down.

u - the letter c but 45 degrees anti-clockwise. But this one is the closest to a proper sided smiley face!

v - It's like :>, but also on the correct side like "u"

w - It's like v, but twice the v's. Also the letter kinda reminds of me of an old person.

x - now THIS is the peak of all the letters, quadruple faces!!!

y - like g, but instead of cool it's just happy u

z - Probably the closest letter to the capital letter variants. Could argue that "e" is closer, but this I would say is closer.
