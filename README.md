# FaceyFont
### Facey Font created for [Typeface Hack club!](https://typeface.hackclub.com/)

![Facey Font Logo](logo/logo.png)
This... is Facey Font!
One of the greatest fonts to every face all... fonts?
The most fonty of fonts to ever font... Alright enough fonting lol.

FaceyFont is basically a font where every single character in the font has a face!
The idea came when I was thinking for an idea to make into a font, and thought about making something new.
That's when making a font with faces came up!

If you're looking for a funny or upgoing font, or even something just nice and simple. Then this is the perfect font for you!

To demonstrate the uses for this font, I've created a website!
This website shows off all of the letters in the font, gives you the ability to write in the font, and real use cases for the font!
If you would like to check out the website, [click here!](https://helloisreal.github.io/FaceyFont/)
